# Customer Churn Analysis
Analysed why telecom customers leave using a dataset of 7,032 records.

## 🔍 Key Findings
- Month-to-month customers churn at **42%** vs 11% on annual contracts
- **First 12 months** is the highest churn risk window  
- Churned customers pay **~$20/month more** on average

## 💡 Recommendation
Offer discounted annual contracts to new customers in their first 3 months.
